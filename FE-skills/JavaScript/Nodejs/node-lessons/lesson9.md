* JS 中需要注意的地方：
    - JS 中，对于四种零宽断言，只支持零宽度正预测先行断言和零宽度负预测先行断言两种
    - JS 中，正则表达式后面可以跟三个 flag：/something/igm
        + i:不区分大小写
        + g:匹配多个
        + m:`^` `$` 可以匹配每一行的开头
