
1、WEB 标准：
	WEB 标准不是某一个标准，而是一系列标准的集合。
	网页主要由三部分组成：结构（structrue）、表现（presentation）、和行为（behavior）。
	结构化标准语言主要包括 XHTML 和 XML；表现标准语言主要包括 CSS；行为标准语言主要包括对象模型、ECMAScript 等。

2、网站架构的流程：
	设计稿的分析；
	切图；
	编写 xhtml 和 css；
	编写 Javascript

3、调用样式的方法比较：
	引用：link
	导入：@import 

4、布局页面的 div id 命名：

	div#container>(div#header+(div#content>div#col_left+div#col_right)+div#footer)
